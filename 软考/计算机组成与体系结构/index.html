    计算机组成与体系结构

    1.计算机结构

    运算器作用：完成算数运算（数值四则）和逻辑运算（与或非）

    运算器：

    1. 算术逻辑单元ALU
    2. 累加寄存器AC
    3. 数据缓冲寄存器DR
    4. 状态条件寄存器PSW(用于存储运算过程中的标志位，比如：进位、溢出、中断等状态信息)

    控制器：

    1. 程序计数器PC(当前指令运行时，用于记录下一条指令的地址位置(顺序则原地址自增即可))
    2. 指令寄存器IR(通过程序计数器PC中的地址取出相应内存中指令后，IR用于暂存这些指令，便于运行)
    3. 指令译码器(用于分析指令的具体操作和操作对象，一条指令可分割为操作码和地址码，所以译码就是通过指令中的这两个码来确定如何做)
    4. 时序部件(用于产生计算机各部件所需定时信号的部件)

    2.计算机体系结构分类--Flynn

    分类依据：

    1. 指令流
    2. 数据流

    阵列处理机：多数用来处理数组的运算

    流水线的吞吐率(Through Put rate, TP)是指在单位时间内流水线所完成的任务数量或输出的结果数量

    5.计算机层次化存储结构

    5.1层次化存储结构

    1. CPU有运算器、控制器，运算器与控制器中有对应的寄存器，寄存器容量极小、速度极快

    2. Cache：高速缓存存储器

    3. 内存
    4. 外存

    速度自下往上增加、容量自上往下增加

    Cache中所有的内容均来自内存

    6.局部性原理

    时间局部性 -> 刚刚访问完的指令，短时间内再次访问(比如：for循环)

    空间局部性 -> 一旦程序访问了某个存储单元，在不久之后，其附近的存储单元也将被访问(比如：数组)

    工作集原理/工作集理论 -> 把要被频繁访问的页面集合打包起来，并放到Cache使之短时间不被替换，达到高效的目的。

    8.总线

    8.1 系统总线

    系统总线包含三总不同功能的总线：

    1. 数据总线DB(Data Bus)
    2. 地址总线AB(Address Bus)
    3. 控制总线CB(Control Bus)

    常见的总线：

    • ISA(Industria Standard Architecture)总线，也叫AT总线

    • EISA总线，在两条ISA信号线之间添加一条EISA信号线。

    • PCI总线由Intel公司推出的一种局部总线

    • 前端总线
    • RS-232C
    • SCSI总线，不属于系统总线
    • SATA
    • USB
    • IEEE-1394
    • IEEE-488

    9.存储器

    9.1 读/写存储器、只读存储器

    按存储器的工作方式可以分为读/写存储器和只读存储器

    1. 读/写存储器(RAM)。既能读取数据也能存入数据
    2. 只读存储器。工作过程中仅能读取。根据数据的写入方式，这种存储器又可细分为ROM、PROM、EPROM、EEPROM
    3. 固定只读存储器(ROM)。内容只能读出，不能改变
    4. 可编程的只读存储器(PROM)。用户可一次性写入，写入后无法修改。
      1. 可擦除可编程的只读存储器(EPROM)。内容可以读出，也可以由用户写入，写入后可以修改
      2. 电擦除可编程的只读存储器(EEPROM)。与EPROM相似，只不过是用电擦除的方法进行数据的改写
      3. 闪速存储器。简称山村，特性介于EPROM与EEPROM之间。整块闪存可以在数秒内删除，速度比EPROM快

    9.2 按地址访问的存储器、按内容访问的存储器

    按访问方式可分为按地址访问的存储器和按内容访问的存储器

    • 按地址访问可分为随机存储器(RAM)、顺序存储器(SAM)和直接存储器(DAM)

    • 按内容访问的有相联存储器
