    变量和字符串

    变量

    Python中的变量区分大小写，命名规范与java相同，数字不能开头

    在Python中，两个变量之间进行数据交换，可以下为一下形式

    x = 3
y = 5
print(x, y)  # 3 5
x, y = y, x
print(x, y)  # 5 3

    字符串

    Python中有三种表示字符串的形式

    1. Single quotes(单引号)
    2. Double quotes(双引号)
    3. Triple quoted(长字符串)

    这样设计可以避免一些问题

    原始字符串

    在前面加r

    print(r"D:\three")

    长字符串

    成对出现

    """ """
''' '''

    字符串的乘法复制

    print("123\n" * 3)
# 123

    数字类型

    整数integers

    浮点数float point numbers

    python中用科学计数法表示小数位超过5位的浮点数

    # 5e-05
print(0.00005)

    复数complex numbers

    x = 1+2j
print(x.real)   # 实部
print(x.imag)   # 虚部

    运算符

    x // y：取比目标结果小的最大整数（向下取整/向小的那个值取整）

    print(3 // 2)   # 1
print(-3 // 2)  # -2

    短路逻辑和运算符优先级

    短路逻辑的核心思想:从左往右,只有当第一个操作数的值无法确定逻辑运算的结果时，才对第二个操作数进行求值。

    内置函数

    divmod()

    返回（x // y, x % y)

    print(divmod(-3, 2))    # (-2, 1)

    abs()

    返回绝对值，如果参数是复数则返回复数的模

    print(abs(1 + 2j))  # 2.23606797749979

    complex()

    将参数转换为复数，参数中不能带空格

    print(complex(1 + 2j))  # (1+2j)

    pow()

    用法与幂运算符**相似

    但pow可以传三个参数，第三个参数作用将幂后的值与第三个参数进行%运算

    即x ** y % z

    bool()

    直接给出参数中的结果(True/False)

    模块

    random模块

    random() 方法返回随机生成的一个实数，它在[0,1)范围内。

    import random

print(random.randint(1, 10))  # 产生 1 到 10 的一个整数型随机数  
print(random.random())  # 产生 0 到 1 之间的随机浮点数
print(random.uniform(1.1, 5.4))  # 产生  1.1 到 5.4 之间的随机浮点数，区间可以不是整数
print(random.choice('tomorrow'))  # 从序列中随机选取一个元素
print(random.randrange(1, 100, 2))  # 生成从1到100的间隔为2的随机整数

a = [1, 3, 5, 6, 7]  # 将序列a中的元素顺序打乱
random.shuffle(a)
print(a)

    decimal模块

    为什么使用？

    import decimal

print(0.1 + 0.2)    # 0.30000000000000004

a = decimal.Decimal('0.1')
b = decimal.Decimal('0.2')
print(a + b)    # 0.3
