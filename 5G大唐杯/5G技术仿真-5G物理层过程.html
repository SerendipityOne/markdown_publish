    5G技术仿真-5G物理层过程

    首先是按顺序选择

    其次，在每一块中都有内容需要配置

    准备

    SSB映射

    出现线条表示选对

    无线帧配置

    无线帧与子帧都是10ms，选择

    子帧需要看前置条件

    子载波间隔为30khz，根据表格，得到时隙为2

    一个时隙有14个符号

    上下行时隙根据前置条件选择

    子帧配置为2.5ms双周期。2.5ms双周期帧结构，每5ms里面包含5个全下行时隙,三个全上行时隙和两个特殊时隙。Slot3和Slot7为特殊时隙，配比为10:2:2（可调整），整体配置为：DDDSUDDSUU，其中：第一个2.5ms为DDDSU,第二个2.5ms为DDSUU;

    特殊时隙配置

    特殊时隙四元组{，9， ，2}

    选择9个D，2个U

    频段 频率
    n41 2.6GHz
    n77 3.4GHz
    n78 3.5GHz
    n79 4.9GHz

    GSCN号根据N、M，再看表二中的公式，根据频段为n41，得到2.6GHz对应0~3GHz选第一条公式。n3可以看做是3，于是得到结果为2107*3=6321

    SSB中心频点，从表二中的第一条公式中的中间那个得出(SS block frequency position)，算得2528550

    SSB绝对信道号，根据表一，n41对应0\3GHz，看△FGlobal~为5，根据SSB中心频点 2528550 / 5=505710

    PBCH公用天线端口号：4k

    SSB时频资源基本配置

    根据官方定义知道，Case C 会使用到4个时隙，所以选取0 1 2 3四个时隙，然后每个SSB固定使用4个OFDM符号

    需要根据之前的SSB中心频点与表格中的Point A来计算： 比如计算出的结果为27，那么就是从17-36的连续20个频率域

    根据SSB频点为2528550，PointA频点为2518470，根据公式（SSB 频点 - PointA 频点）/（12*30）= 28

    28 是 SSB 的第十（从 0 开始数）个 RB 的位置，向前数 10 个即为SSB 起点，即从18~37共20个RB

    结束后可以点击结果校验，绿色为正确

    检PSS/SSS

    根据前置条件PCI为154，

    PSS固定\(N^2_{ID}\)，值是PCI % 3 = 1

    SSS固定\(N^1_{ID}\)，值是PCI / 3 = 51

    PSS是绿色，SSB是红色，占用频域资源是固定的，56-182

    解DMRS

    PCI=154，v=PCI % 4=2，找出所有 % 4 = 2的进行填充，两边需要填充三列，中间只需填充第一、第三列。

    第二行不填的范围是

    2→48~191

    检PBCH

    将剩下的全部填充即可，不填范围也是一样2→48~191

    读MIB

    查表三

    根据第十行，最后一列得知SSB 向前偏 12 个 RB 为CORESET 0 的位置，由前面模块得知SSB起始位置为CRB18，则填充CRB 6 ~ CRB 53（第三列为48，共填充48个）
